translationCore builds

Read Code Documentation.

Automated builds for translationCore can be found at the below links.

  • tc-develop.unfoldingWord.surge.sh develop branch
  • tc-sandbox.unfoldingWord.surge.sh sandbox-* branches
  • tc-release.unfoldingWord.surge.sh release-* branches

Builds will only trigger if all tests pass.

Builds are are not triggered on pull requests.

Builds will not be created for the master branch.

Builds are generated for tags but they are instead uploaded directly to releases on github.
